протокол взаимодействия с модулем m-storage

m-storage исполнять следующие команды по обработке индивидов приходящие ему в сообщении по протоколу nanomsg:

"put" - сохранить индивид
"remove" - удалить индивид
"add_to" - добавить (поле/поля) в индивид 
"set_in" - Установить (поле/поля) в индивид 
"remove_from" - Удалить (поле/поля) из индивида 


формат JSON сообщения:

{
    "function": /команда в виде строки/,
    "ticket": /идентификатор сессионного билета/,
    "individuals": /[массив индивидов в виде обьектов]/,
    "assigned_subsystems": /ОПЦИОНАЛЬНО: 64 bit число, если 0 то сообщение предназначается к обработке для всех модулей/  
    "event_id" : /ОПЦИОНАЛЬНО: инкрементальный идентификатор цепочки скриптов вследствии работы которых был сгенерерован данный индивид/индивиды/,
    "src" : /ОПЦИОНАЛЬНО: идентификатор группы скриптов/,
    "addr" : /ОПЦИОНАЛЬНО: ip адрес клиента, если указан то будет сопоставлятся с ip адресом из ticket/
}
